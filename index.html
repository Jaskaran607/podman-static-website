📘 What is Podman?

Podman is a container engine developed by Red Hat. It's a Docker-compatible tool that allows you to manage containers without a central daemon and supports rootless containers.

🔧 Useful Podman Commands

# Check version
podman --version

# Pull an image
podman pull nginx

# Run a container
podman run -d -p 8080:80 nginx

# List containers
podman ps

# Stop container
podman stop <container_id>

# Remove container
podman rm <container_id>

🌐 HTML/CSS Web Project with Podman

  1. Create a folder with index.html, style.css, and Dockerfile.
  2. Sample Dockerfile:
FROM nginx:alpine
COPY index.html /usr/share/nginx/html/index.html
COPY style.css /usr/share/nginx/html/style.css

Build and run using Podman:

podman build -t podman-html-site .
podman run -d -p 8080:80 podman-html-site

💡 Tips for Podman Users

  • No daemon needed (unlike Docker)
  • Run as rootless user: alias docker=podman
  • Compatible with Dockerfiles and Compose (with podman-compose)
  • Use volumes for persistent data: -v ./data:/data
